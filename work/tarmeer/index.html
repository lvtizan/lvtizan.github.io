Tarmeer

多国家 · 多语言室内设计垂直平台

项目概述

从零重建的多国家、多语言室内设计垂直平台，打通「业主找装企 / 专家 / 供应商 + 作品集展示 + 外勤问卷 + 招商」全链路，服务阿联酋（迪拜）与越南两国，严格数据隔离。技术栈：Next.js（App Router · SSR）+ TypeScript · Express · MySQL（阿里云 RDS 迪拜）· PM2 · Nginx · 自建 ONNX/CLIP 图像标注。
